# How to Use

```
./windows-safe-pipe-386.exe -h
NAME:
   windows-safe-pipe - CLI tool to echo windows safe pipe scripts

USAGE:
   windows-safe-pipe [global options] command [command options]

VERSION:
   0.1.3

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --action value   support action: read / write / exit
   --pipe value     pipe name
   --message value  the write message
   --help, -h       show help
   --version, -v    print the version
```
