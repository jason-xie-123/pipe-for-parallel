# How to Use

```
./pipe-for-parallel -h
NAME:
   pipe-for-parallel - CLI tool to pipe-for-parallel scripts

USAGE:
   pipe-for-parallel [global options] command [command options]

VERSION:
   0.1.3

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --action value   support action: read / write / exit
   --pipe value     pipe name
   --message value  the write message
   --help, -h       show help
   --version, -v    print the version
```
